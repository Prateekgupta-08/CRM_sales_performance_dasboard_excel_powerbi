# 📊 CRM Sales Performance Analysis | Power BI

## Project Overview
Developed an interactive **CRM Sales Performance Dashboard** in **Power BI** to help sales managers monitor business performance, analyze sales trends, and identify growth opportunities through data-driven insights.

## Actual Client Problem:
B2B sales pipeline data from a fictitious company that sells computer hardware, including information on accounts, products, sales teams, and sales opportunities. They have just started using a new CRM system to track their sales opportunities but have no visibility of the data outside of the platform. So I created this Dashboard for them.

## 🎯 Business Objective

Transform CRM sales opportunity data into an interactive dashboard that enables stakeholders to:

- Track overall sales performance
- Monitor Win vs Loss ratio
- Analyze quarterly sales growth
- Evaluate sales agent performance
- Compare product and regional performance
  
---
## 🚀 Key KPIs

- 💰 $10M+ Closed Deal Value
- 🤝 4,238 Won Deals
- 📈 63% Win Rate
- 💵 Average Deal Size Analysis
- 📊 Quarter-over-Quarter Growth
- 🌍 Regional Performance
- 🏆 Top Sales Agents

## 💡 Key Insights

- Achieved a **63% opportunity win rate**.
- Identified top-performing sales agents and regions.
- Compared product performance using revenue, deal count, and average deal size.
- Tracked quarterly sales growth using dynamic KPI scorecards.
- Built an interactive dashboard with slicers for deeper business analysis.
